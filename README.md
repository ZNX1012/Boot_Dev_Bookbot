This is a project for a Bookbot, as described on <https://www.boot.dev>.\
Specifically, the course <https://www.boot.dev/courses/build-bookbot-python>.\

Bookbot is able to take a given book in .txt format and return the following statistics:
 - number of words in the book
 - a sorted list with the corresponding number of alphabetical letters
    - sorted in descending order

Usage:
python3 main.py ../books/<path_to_book> OR any <path_to_book> in .txt format
